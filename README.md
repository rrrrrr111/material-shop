
#Команды
    `build` - Сборка бэкенда
    `Ctrl + F9` - При запущеном приложении со spring-boot-devtools, сборка в IDEA перезапускает приложение

#Первичная настройка
    1. IntelliJ IDEA или др. IDE для работы с Java
